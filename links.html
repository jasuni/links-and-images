This example is about links and images.

This particular page is about links. There are 3 kinds of links:

  • Absolute links, which can go anywhere. For example, Mozilla Developer Network, which is supposed to be a useful resource for web developers.
  • Relative links start from the current location, so they are useful for moving to our own extras page
  • Root-relative links are useful on websites, as they allow us to create a reference starting from the root of the website. It is a rather easy way to create a link to the home page of a website.
