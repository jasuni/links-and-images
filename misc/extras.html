Character Sets

Using UTF-8 expands the character set you can use. The following list requires UTF-8

  1. bir
  2. iki
  3. üç
  4. dört
  5. beş

Reserved Characters

There are three special characters for HTML: < > and &. for these characters, you use an HTML entity, which begins with an & and ends with a ;
